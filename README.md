## 🚀 Step-by-Step Deployment Guide for GoDaddy

### Prerequisites
✅ GoDaddy account with domain **www.abhishekchauhan.co.in** purchased
✅ All website files (index.html, css/, js/, images/)
✅ FTP client (FileZilla recommended) OR GoDaddy File Manager access

---

### Method 1: Using GoDaddy File Manager (Recommended for Beginners)

#### Step 1: Log in to GoDaddy
1. Go to [www.godaddy.com](https://www.godaddy.com)
2. Click **Sign In** (top right)
3. Enter your GoDaddy username/email and password

#### Step 2: Access cPanel or Web Hosting Dashboard
1. From your account dashboard, click **Web Hosting**
2. Find your hosting plan for **abhishekchauhan.co.in**
3. Click **Manage** button
4. Look for **cPanel** or **Plesk** access (usually a button labeled "cPanel Admin" or "Admin Panel")

#### Step 3: Open File Manager
1. In cPanel, find and click **File Manager** (under Files section)
2. Navigate to **public_html** folder (this is your website root)
3. If there are any default files (like `index.html`, `coming_soon.html`), select and **delete** them

#### Step 4: Upload Website Files
1. Click **Upload** button (top of File Manager)
2. Upload the following files from your portfolio:
   - `index.html`
   
3. After uploading `index.html`, click **Go Back to File Manager**

4. Now create folders and upload subdirectory files:
   - Click **+ Folder** button, create folder named `css`
   - Enter the `css` folder
   - Click **Upload**, upload `style.css` file
   - Go back to `public_html`
   
   - Click **+ Folder** button, create folder named `js`
   - Enter the `js` folder
   - Click **Upload**, upload `script.js` file
   - Go back to `public_html`
   
   - Click **+ Folder** button, create folder named `images`
   - Enter the `images` folder
   - Click **Upload**, upload all 6 image files:
     * `abhishek-chauhan.jpg`
     * `cyber-security.jpg`
     * `green-energy.jpg`
     * `manufacturing.jpg`
     * `fintech.jpg`
     * `banking-nbfc.jpg`
   - Go back to `public_html`

#### Step 5: Set File Permissions
1. Right-click on `index.html`, select **Change Permissions**
2. Set to **644** (Read and Write for Owner, Read for Group and Public)
3. Do the same for all CSS and JS files

#### Step 6: Test Your Website
1. Open a new browser tab
2. Go to **www.abhishekchauhan.co.in**
3. Your portfolio should now be live! 🎉

### Method 2: Using FTP Client (FileZilla)

#### Step 1: Download FileZilla
1. Visit [https://filezilla-project.org/](https://filezilla-project.org/)
2. Download and install FileZilla Client (free)

#### Step 2: Get FTP Credentials from GoDaddy
1. Log in to GoDaddy account
2. Go to **Web Hosting** → **Manage**
3. Find **FTP** section (or look for "FTP/SFTP" in settings)
4. Note down:
   - **FTP Hostname**: Usually `ftp.abhishekchauhan.co.in` or an IP address
   - **FTP Username**: Your GoDaddy username or provided FTP username
   - **FTP Password**: Your FTP password (you may need to reset it)
   - **Port**: Usually `21` for FTP or `22` for SFTP

#### Step 3: Connect with FileZilla
1. Open FileZilla
2. Enter credentials in top bar:
   - **Host**: Your FTP hostname
   - **Username**: Your FTP username
   - **Password**: Your FTP password
   - **Port**: 21 (or 22 for SFTP)
3. Click **Quickconnect**

#### Step 4: Upload Files
1. In FileZilla:
   - **Left side** = Your local computer files
   - **Right side** = GoDaddy server files
2. On right side, navigate to `/public_html/` folder
3. Delete any default files in `public_html`
4. On left side, navigate to your portfolio folder
5. Select all files and folders:
   - `index.html`
   - `css/` folder
   - `js/` folder
6. **Drag and drop** from left to right to upload
7. Wait for upload to complete (green checkmarks)

#### Step 5: Verify Upload
1. In FileZilla right panel, confirm you see:
   ```
   public_html/
   ├── index.html
   ├── css/
   │   └── style.css
   └── js/
       └── script.js
   ```

#### Step 6: Test Website
1. Go to **www.abhishekchauhan.co.in** in your browser
2. Your portfolio is now live! 🎉

## 🔧 Troubleshooting

### Website Not Showing
- **Clear browser cache**: Press `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
- **Check DNS propagation**: May take 24-48 hours for domain to fully propagate
- **Verify file location**: Ensure `index.html` is directly in `public_html`, not in a subfolder

### CSS/JS Not Loading
- **Check file paths**: Ensure folder structure is correct (`css/style.css`, `js/script.js`)
- **Check file permissions**: Should be `644` for files, `755` for folders
- **Check browser console**: Press `F12` → Console tab for errors

### Domain Shows GoDaddy Parked Page
- **DNS Settings**: In GoDaddy dashboard, go to DNS Management
- **A Record**: Ensure A record points to your hosting IP
- **CNAME**: Ensure www points to your domain
- **Wait**: DNS changes can take up to 48 hours

### Images Not Loading
- This portfolio uses CDN-based icons (Font Awesome) and fonts (Google Fonts)
- Ensure internet connection is active for these resources to load
- Check browser console (F12) for any blocked resources

## 🔒 Security Recommendations

1. **HTTPS Certificate**: Enable SSL through GoDaddy (usually free with hosting)
   - In GoDaddy cPanel → SSL/TLS → Install SSL Certificate
   
2. **File Permissions**:
   - Files: `644` (rw-r--r--)

3. **Regular Backups**:
   - Use GoDaddy's backup feature
   - Download local copies periodically

4. **Password Security**:
   - Use strong FTP/cPanel passwords
